# YEMEKİMNEREDE

---

## Proje Hakkında

![Ürün Tanıtım Görseli](Product.png)

**Proje Tanımı:**
YEMEKİMNEREDE, kullanıcıların çevrelerindeki yemek mekanlarını kolayca keşfedebileceği bir uygulamadır. Kullanıcılar konumlarına göre yakındaki restoranları listeleyebilir, anlık sipariş takibi yapabilir ve kişiselleştirilmiş öneriler alabilir. "Yemeğinizi saniyeler içinde bulun" mottosuyla yemek bulma deneyimini yeniden tanımlıyoruz.

**Proje Kategorisi:**
E-Ticaret

**Referans Uygulama:**
